Release Notes |version|
=======================

This release provides an implementation of the OpenStack Neutron LBaaSv2 driver and agent to support F5 Networks® BIG-IP® systems.

Release Highlights
------------------

This release introduces support for SSL offloading and for BIG-IP® clustering (active-standby and scalen configurations).
