Summary
-------

This release provides an implementation of the F5® service provider driver to support the use of F5 Networks® BIG-IP® systems with OpenStack Neutron LBaaSv2.

Release Highlights
------------------

This release introduces the following:

- Support for Cisco ACI opflex networks
- Bug fixes

See the `changelog <https://github.com/F5Networks/f5-openstack-lbaasv2-driver/compare/v9.2.0...v9.3.0.b1>`_ for the full list of changes in this release.
