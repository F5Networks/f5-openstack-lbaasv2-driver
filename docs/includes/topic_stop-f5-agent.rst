Stop the F5 OpenStack agent
===========================

If you need to stop the F5 agent, run the command appropriate for your OS.

.. topic:: CentOS

    .. code-block:: shell

        $ sudo systemctl stop f5-openstack-agent.service


.. topic:: Ubuntu

    .. code-block:: shell

        $ sudo service f5-oslbaasv2-agent stop
